## Install and run

### Install

1. Create venv:

```sh
python -m venv venv
```

2. Source activate script:

```sh
source venv/bin/activate
```

3. Install requirements:

```sh
pip install -r requirements.txt
```

4. Copy .env:

```sh
cp .env.example .env
```

5. Fill .env variables.

### Run

Run `thesis.ipynb` notebook.
